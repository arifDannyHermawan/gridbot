Grid Bot Guide

📌 What is DCA?

Dollar Cost Averaging: buying assets periodically with fixed amounts. Reduces volatility & improves average entry price.

🎯 Features

💰 Multi Bot: manage multiple coins
📊 Range: auto calculate target price
🔒 Lock: freeze coin data
✓ DCA Marker: tap row to mark done
📥 Backup: export/import data

📖 Step-by-Step

1️⃣ Set GLOBAL PARAMETERS (entry count, drop/rise %)
2️⃣ Tap "+ Add Asset" for new coin
3️⃣ Fill LONG & SHORT: nominal & amount/entry
4️⃣ Tap table row to mark "purchased" (highlight=done)
5️⃣ Lock coin 🔒, export backup via menu ⋯

💡 Tips

• LONG = uptrend, SHORT = downtrend strategy
• Min Price (LONG) = lowest buy; Max Price (SHORT) = highest sell
• Lock coin when final (prevent accidental edit)
• Regular backup to safe location
• Tap DCA table row to track execution progress

📚 Glossary

Entry: Each buy order
Drop %: % price drop for next entry (LONG)
Rise %: % price rise for next entry (SHORT)
Cumul: Cumulative nominal invested
Avg Price: Average entry price of all orders
Lock: Freeze coin (read-only)
